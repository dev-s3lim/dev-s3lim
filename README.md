## 주요 프로젝트

### 누리다(NURIDA) 프로젝트
- 롯데호텔 협력사의 세탁 업무 자동화 시스템 기획 및 설계
- 기존 수기 기반 접수/분류/배송 절차를 앱 및 PC 기반 시스템으로 전환
- 음식 배달 앱과 같이 호텔 고객의 세탁 요청부터 픽업, 분류, 딜리버리 현황까지 실시간 관리 가능
- 휴먼 에러 감소 및 전체 작업 시간 단축 → **롯데호텔과의 재계약 성사에 실질적 기여**

### Moodmend 명상 플랫폼
- 감정 기반 콘텐츠 추천, 유저 아바타, 인앱 스토어 등 복합 기능 포함 명상 서비스 설계
- 회원, 콘텐츠, 감정, 예약, 결제, 커뮤니티 등 12개 이상 테이블 직접 ERD 설계
- 기능별 프로시저(DELIMITER) 기반 구조 설계 및 트랜잭션 흐름 정의
- 기획부터 데이터 설계, 서비스 로직, 유지보수까지 고려한 실현 가능한 플랫폼 구축
